This is a responsive web based slot booking tool.

When you run the project the database will be created and some test data should be automatically generated into 
the database from '01-init.sql'.
If not, please run this SQL code manually. If using VSCode, this can be done by opening the '01-init.sql' file
and typing ctrl + shift + p and running an SQLite query. 

To run this project please run these commands in the terminal:

1) 'npm install'
2) 'node app.js'

The server will be hosted on 'localhost:5000'
